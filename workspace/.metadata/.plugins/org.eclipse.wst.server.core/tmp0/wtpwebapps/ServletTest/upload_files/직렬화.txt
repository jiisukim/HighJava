직렬화 
- 값 형식 데이터 (int,float,char) 등의 값 형식 데이터들은 스택에 메모리 쌓이고
직접 접근이 가능하다
- 참조형식 데이터는 변수를 선언하면 힙에 메모리가 할당되고 스택에서는 이 힙 메모리를 참조하는 구조로 되어있다
참조형식의 데이터는 실제 데이터 값이 아닌 힙에 할당되어있는 메모리 번지 주소를
가지고 있기 때문에  저장,통신에 사용할 수 없다.
그래서 직렬화를 통해 각 주소값이 가지는 데이터들을 끌어모아 값 형식의 데이터로
변환해준다.


값 형식 데이터 : "" 직접 접근 "" O
참조형식 데이터: "" 직접 접근 "" X
